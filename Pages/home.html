Description of Pages

Map

Allows the user to automatically find their location and report a stranding based on that location. Also shows all strandings

as an individual icon on the map. Clicking an icon shows the related mammal and location for the selected stranding.

Report Stranding

Allows the user to report a stranding by manually entering attributes (longitude and latitude can be automatically populated).

Organization

Allows the user to add a new responder to the organization by manually entering attributes. This page also allows the user to

update existing users or delete existing users from the database.

Strandings

Allows the user to search the strandings table filtered by the responder that was involved (drop down with responders will be

dynamically populated from the responders table), update a stranding by ID, search mammals associated with a stranding ID,

and add a responder to a specific stranding.
